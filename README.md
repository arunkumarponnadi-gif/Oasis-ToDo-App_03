# 📋 To-Do Web Application

A modern and responsive To-Do Web Application built using HTML, CSS, and JavaScript. This project was developed as part of the AICTE Oasis Infobyte Web Development Internship (Level 2 - Task 3).

## 🚀 Project Overview

The To-Do Web Application helps users efficiently manage daily tasks. Users can add tasks, mark them as completed, view pending and completed tasks separately, and delete tasks whenever required. The application provides a clean and user-friendly interface for better productivity and task organization.

## ✨ Features

- ➕ Add new tasks instantly
- ✅ Mark tasks as completed
- 📌 Separate Pending Tasks section
- 🎯 Separate Completed Tasks section
- 🗑️ Delete tasks easily
- 📱 Responsive design for all devices
- ⚡ Fast and lightweight performance
- 🎨 Clean and intuitive user interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
